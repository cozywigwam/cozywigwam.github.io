* Syntax

** Maps

Sorta like JS objects

#+BEGIN_SRC clojure
{:id 1 :name "test"}
#+END_SRC

** Vectors

0-indexed array

#+BEGIN_SRC clojure
[1 2 3]
#+END_SRC

** Lists

Requires n iterations to get a value

#+BEGIN_SRC clojure
'(1 2 3)
#+END_SRC


** Sets

Contain unique values

#+BEGIN_SRC clojure
#{:a "apple" :b "banana"}
#+END_SRC
